A testimonials plugin to display client testimonials or quotes on your page or posts.

== Description ==

Display testimonials in speech bubbles on your pages or posts. Can also include an author/image.

Useful for

*   Showing feeback from clients or customers
*   Famous Quotes
*   Anything that requires a speech bubble

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[tesimonial author="me"] I said this [/testimonial]` in your templates
